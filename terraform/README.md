### Deploy

**Clone the project**
```
$ git clone https://github.com/tedleyem/atlassian
```

**Move into directory**
```
$ cd terraform
 ```

**Setup environment: INIT**
```
$ terraform init
```
 **Configure environment: PLAN**
```
$ terraform plan
```

 **Deploy: Apply**
```
$ terraform apply
```
